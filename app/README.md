# MyApp (Expo)

This app now runs with Expo.

## Run the project

```sh
npm install
npm start
```

## Open on devices

- Android: `npm run android`
- iOS: `npm run ios`
- Web: `npm run web`

## Notes

- `npx install-expo-modules@latest` was used to migrate dependencies and native integration.
- On Linux, iOS build tooling is not available. Run iOS locally on macOS if needed.
